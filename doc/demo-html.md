## Input Attrs

- disabled	规定输入字段应该被禁用。
- max	规定输入字段的最大值。
- maxlength	规定输入字段的最大字符数。
- min	规定输入字段的最小值。
- pattern	规定通过其检查输入值的正则表达式。
- readonly	规定输入字段为只读（无法修改）。
- required	规定输入字段是必需的（必需填写）。
- size	规定输入字段的宽度（以字符计）。
- step	规定输入字段的合法数字间隔。
- value	规定输入字段的默认值。
- autocomplete 属性规定表单或输入字段是否应该自动完成。
- autofocus 自动获得焦点
- form 属性规定inpu元素所属的一个或多个表单。
